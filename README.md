# StructToGraph
  Convert structural analysis models (Abaqus INP, YJK INP/DAT) into PyTorch Geometric graph data for GNN-based dynamic response prediction. Supports          beam/column topology extraction, shell element aggregation, modal analysis integration, and ground motion response alignment.   将Abaqus/YJK结构模型转换为PyG图数据，支持梁柱拓扑提取、剪力墙聚合、模态分析与地震响应数据集成。
